# Installation Commands

## Root

```bash
npm install
```

## Frontend Dependencies

```bash
npm install -w apps/frontend react-router-dom zustand axios @tanstack/react-query tailwindcss clsx lucide-react
```

## Backend Dependencies

```bash
npm install -w apps/backend express mongoose dotenv cors jsonwebtoken bcryptjs zod morgan helmet
```

## App Dev Dependencies

```bash
npm install -D -w apps/frontend @types/react @types/react-dom @vitejs/plugin-react vite typescript eslint prettier
npm install -D -w apps/backend @types/express @types/node @types/cors @types/jsonwebtoken @types/bcryptjs @types/morgan tsx nodemon typescript eslint prettier
```

## PM2 (Global)

```bash
npm install -g pm2
```
